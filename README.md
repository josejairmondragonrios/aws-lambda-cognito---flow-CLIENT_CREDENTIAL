## AWS Cognito Lambda nodejs
AWS Cognito - Lambda nodejs
- flow client_credential.

### 1- Install packages
```bash
npm install
```

### 2- Create Environment .env
```bash
AppClientId=
AppClientSecret=
UrlCognitoOauth2Token=
```

### Run index.js - local
```bash
npm run index
```

### Run getToken.js - local

```bash
npm run signUp
```
